# E-commerce Sales Analysis Dashboard (Tableau)

## Project Overview
End-to-end Tableau dashboard analysing e-commerce sales performance, profitability, discounts, delivery, and customer segments for CEO decision-making.

**Primary Objective**: Identify key drivers of profitability and uncover opportunities to improve margins.

## Business Questions Answered
1. Are top-selling products the most profitable?
2. How do discounts and shipping costs impact profitability?
3. Which segments and cities drive revenue and profit?
4. Do delivery delays and cancellations affect ratings?

**Key Insights & 5 Business Recommendations** are available in the documentation file.
